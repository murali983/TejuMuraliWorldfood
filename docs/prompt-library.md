# AI Prompt Library

## Universal food image base prompt
```text
Luxury food magazine photography of [RECIPE NAME], [CUISINE] cuisine, editorial natural light, realistic food texture, premium ceramic plating, soft shadows, shallow depth of field, clean table styling, no text, no watermark, no logo.
```

## Universal short-form video base prompt
```text
Create a vertical 9:16 cooking video for [RECIPE NAME] with a 3-second hook, close ingredient detail, fast but readable step transitions, hand movement shots, steam or texture reveal, and a warm expert voiceover script.
```
